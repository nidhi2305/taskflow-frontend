## Tech Stack

- React (Vite)
- JavaScript
- Tailwind CSS
- Axios
- React Router DOM

---

## Features

- User authentication (Register & Login)
- JWT-based protected routes
- Dashboard with task statistics
- Create, edit, delete tasks
- Task status and priority management
- Search and filter tasks
- Responsive UI
